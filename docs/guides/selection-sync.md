---
title: Selection Sync
---

When enabled, selecting Scene Object nodes can select referenced viewport objects, and selecting objects can select matching Scene Object nodes.

## Steps

1. Add the relevant preset from the Scene Graph Nodes Add/Search menu when available.
2. Assign source and target objects.
3. Inspect the links and node properties.
4. Run **Evaluate**.
5. Add **Watch Value** nodes when you need to confirm intermediate values.
