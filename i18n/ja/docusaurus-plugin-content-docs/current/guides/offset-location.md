---
title: Offset Location
---

Vector Value と Vector Math Add を使って、コピーした location に offset を加えてから書き込みます。

## 手順

1. 利用できる場合は Scene Graph Nodes の Add/Search メニューから preset を追加します。
2. source / target object を設定します。
3. リンクとノードプロパティを確認します。
4. **Evaluate** を実行します。
5. 中間値を確認したい場合は **Watch Value** を追加します。
