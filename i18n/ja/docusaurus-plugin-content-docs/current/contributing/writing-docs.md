## ルール

- 英語ページと日本語ページを同期します。
- ノードごとに 1 ページを追加します。
- 公開後の slug は安定させます。
- ノード class の追加、rename、削除時は `data/nodes.json` を更新します。
- 実際のノードチェーンに対応する短い例を優先します。

## ページテンプレート

すべてのノードページには purpose、inputs、outputs、properties、evaluation behavior、example、limitations、`node_id`、`node_class`、`added` front matter を含めます。
